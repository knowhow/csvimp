Release Notes
xTuple CSV Import
Version 0.4.3
November 1, 2010
==================================

This is a release of xTuple's CSV Import tool (CSVImp) including
internal architectural changes, bug fixes, and new features.

The following features and bug fixes have been added to the
applications since the release of 0.4.2. Additional detail for each
item listed below may be found on our community website (www.xtuple.org).
Simply go to the Issue Tracker, choose the CSVImp project, and
select the "Changelog" option.


New Features:

N/A

Bug Fixes:

* Fixed crash problem with embedded plugin when windows closed in specific
order
* Field type corrected for text in field mappings window to show length when
possible
* Fixed crash caused from incomplete initialization
* Fixed File | Exit option to only close CSVimp windows when embedded
